# James PIE

Simple SMS communicating task running. The main idea behind this program is to be able to have a Raspberry Pi always running and Wake-On-Lan(WoL) my Windows 10 machine through SMS. James will be developed to run other task in the future..

## Task List
1. WoL Home computers by sending: "James, can you wake up my computer?"
